Static fallback · 第 3 课

single controller 混合控制器

能访问 CDN 时切换为交互版。本课讲清 verl（HybridFlow）最具区分度的设计：单控制器（single controller）如何用一个驱动进程（driver）像调用本地对象一样调用一整组分布在多张 GPU 上的 worker。核心机制是 Worker 基类、WorkerGroup 与 ResourcePool 抽象、@register 装饰器与 dispatch / execute / collect 三段式，以及 RayWorkerGroup 如何用 Ray actor 实现这一切并把多个角色共置（colocate）到同一组 GPU。
