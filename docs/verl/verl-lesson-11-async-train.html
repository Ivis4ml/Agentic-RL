异步训练范式

能访问 CDN 时切换为交互版。本课回答一个被反复追问的问题：verl 的异步训练究竟实现了没有。结论是已经实现，位于 verl/experimental 与 verl/trainer/ppo/v1，并分为多种范式：one_step_off_policy（陈旧度恰好一步的重叠）、fully_async_policy（rollouter 与 trainer 经 message_queue 解耦的全异步）、separation（训练与推理物理分离）、以及 v1 家族的 trainer_sync / trainer_colocate_async / trainer_separate_async（由 transfer-queue 驱动的 agent loop 与 replay buffer 支撑）。本课逐文件精读这些源码并给出取舍表。
