Async 推理服务与 server 模式

能访问 CDN 时切换为交互版。本课讲清 verl 如何把推理引擎包装成常驻的 OpenAI 兼容 HTTP 服务：RolloutReplica 副本抽象与 LLMServerManager 生命周期管理、GlobalRequestLoadBalancer 的粘性会话与负载均衡路由、vLLM / SGLang / TensorRT-LLM 三种 async server 后端、SGLang 的 prefill/decode（PD）分离、partial rollout 的中断与续写、权重热更新时的 wake_up / sleep / KV 缓存释放，以及 main_generation_server 入口。这套 server 模式是 verl 支持 agentic 与异步 RL 的基础设施。
