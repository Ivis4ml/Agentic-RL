DataProto 数据协议

能访问 CDN 时切换为交互版。本课讲清 verl 中一切数据流动的载体 DataProto：它以 TensorDict 为张量后端、以 numpy 对象数组承载非张量数据、以普通字典承载 meta_info 的三段式结构；concat / chunk / split / select / pop / union / repeat / padding 等批操作在分发（dispatch）时如何切分与聚合；DataProtoFuture 如何与 Ray future 配合实现惰性数据；以及它与 Ray 对象存储之间的序列化传递。
