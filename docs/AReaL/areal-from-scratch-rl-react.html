从 AReaL 到你自己的 RL 训练框架

这个文件是一个 React 单页讲解。如果浏览器能访问 React CDN，会自动切换成完整交互版；否则仍可阅读这份静态摘要。

  • AReaL 的核心分层：Scheduler、InferenceEngine、TrainEngine、Workflow、PPO/GRPO controller。
  • 从零搭建时先做同步 GRPO，再拆 rollout 和 training，最后引入异步 staleness、权重同步和 tree attention。
  • 新版增加了核心算法代码章节：trajectory packing、rollout、staleness、PPO/GRPO/GSPO、decoupled PPO、TrainEngine、权重同步、KV routing、varlen/tree attention。
  • KV cache 建议交给 SGLang/vLLM；训练侧不要用 generation KV cache，使用 packed varlen attention、FlexAttention 或 tree attention 解决前缀重复。
  • A100 优先 FlashAttention 2 + bf16；H100 可以加 FlashAttention 3、FP8、torch.compile、CUDA graph 和更激进的 fused kernels。
