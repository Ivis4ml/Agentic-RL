Gateway 节点与 proxy

能访问 CDN 时切换为交互版。本课讲清 Polar 的 Gateway 层：GatewayNodeManager 的 INIT/READY/RUN/POSTRUN 生命周期、FastAPI server 如何接收 dispatch 并把 proxy 暴露给 agent、SGLangClient 如何坐在 agent 进程与推理服务之间拦截 completion 并捕获生成、以及 session / storage / completion_writer / detection 如何协同记录补全以供轨迹重建。
