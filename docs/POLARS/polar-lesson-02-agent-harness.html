Agent 与 Harness 层

能访问 CDN 时切换为交互版。本课讲清 Polar 如何用一个统一的 BaseHarness 契约描述"一个 agent harness"，如何按 AgentSpec 配置实例化（factory），以及八种真实商用 harness（claude_code、codex、gemini_cli、opencode、openhands_sdk、pi、qwen_code、shell）如何零改造接入：每个 harness 只负责拼出启动命令，gateway 通过注入 ANTHROPIC_BASE_URL / OPENAI_BASE_URL / GOOGLE_API_URL 等环境变量把它们的推理请求重定向到训练中的模型。这正是"把成熟商业 agent 直接当 RL 环境"的范式。
